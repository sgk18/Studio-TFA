Explain binary search trees in simple terms with an example
✦
A Binary Search Tree (BST) is a tree where each node has at most 2 children, and the left subtree contains only nodes with values smaller than the parent, while the right subtree contains only larger values.

Example: Insert 8, 3, 10, 1, 6. The root is 8. 3 goes left. 10 goes right. 1 is smaller than 3 so it goes left of 3. 6 is between 3 and 8 so it goes right of 3. Search is O(log n) average!
